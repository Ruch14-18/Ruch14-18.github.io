-> To-Do List Web App
This is a simple and interactive **To-Do List Web Application** built using HTML, CSS, and JavaScript.  
The project is deployed using **GitHub Pages**.

-> Live Website: 
https://ruch14-18.github.io/

-> Project Objective
To build and deploy a static website using GitHub Pages and demonstrate front-end development skills.

-> Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- GitHub Pages (for deployment)

-> Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive UI
- Instant updates using JavaScript DOM manipulation
